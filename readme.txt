--------------Инструкция по запуску проекта----------------------------------------------------------------------------------------------
-------------Предварительные требования---------------------------------------------------------------------------------------------------
Убедитесь, что у вас установлен Go версии 1.16 или выше.
Установите Git для работы с репозиториями (если ещё не установлен).
Запуск проекта

-bash-
git clone <https://github.com/GutsRS/-.git>
cd calc_service
Запустите сервис с помощью команды:

bash
go run ./cmd/calc_service/...
Сервис будет доступен по адресу:
arduino
Копировать код
https://github.com/GutsRS/-.git
Пример запросов через curl
-------------------------------------------------------------------------------------------------------------------------------------
Успешный запрос:
bash
curl --location 'https://github.com/GutsRS/-.git' \
--header 'Content-Type: application/json' \
--data '{
  "expression": "2+2*2"
}'
---------------------------------------------------------------------------------------------------------------------------------------
Ожидаемый ответ:
json
{
  "result": "6"
}
------------------------------------------------------------------------------------------------------------------------------------
Ошибка 422 (невалидное выражение):
bash
curl --location 'https://github.com/GutsRS/-.git' \
--header 'Content-Type: application/json' \
--data '{
  "expression": "2+2a*2"
}'
-------------------------------------------------------------------------------------------------------------------------------------------------
Ожидаемый ответ:
json
{
  "error": "Expression is not valid"
}
--------------------------------------------------------------------------------------------------------------------------------------------
Ошибка 500 (внутренняя ошибка):
Эта ошибка возникает, если что-то пошло не так на сервере.

Упрощённый запуск через компиляцию
Если вы хотите запустить скомпилированный бинарный файл:

Скомпилируйте проект:

bash
Копировать код
go build -o calc_service ./cmd/calc_service
Запустите бинарный файл:

bash
./calc_service
